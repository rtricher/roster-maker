# Roster Maker - Web App

Desktop roster builder interface for Warhammer 40K armies.

## Features

- Create and manage rosters
- Track unit costs and composition
- Real-time point calculations
- Export rosters (coming soon)

## Development

```bash
# Start development server
pnpm dev

# Build for production
pnpm build

# Preview production build
pnpm preview
```

## Stack

- React 18
- TypeScript
- Vite
- TailwindCSS
- React Router v6
