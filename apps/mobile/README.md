# Roster Maker - Mobile App

Mobile-first game day tracker for Warhammer 40K games.

## Features

- Turn counter
- Real-time statistics tracking
- Player kill point tracking
- Battle notes
- Mobile-optimized interface
- Works on any device with a browser

## Development

```bash
# Start development server
pnpm dev

# Build for production
pnpm build

# Preview production build
pnpm preview
```

## Stack

- React 18
- TypeScript
- Vite
- TailwindCSS (mobile-first)
- React Router v6
